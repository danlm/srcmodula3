Variables

In addition to the entries above, a program or library package may redefine any of the following variables:

M3
The Modula-3 compiler to use (default /lib/m3).

M3OPTIONS
The primary compilation flags (default -w1 -why -g). Use m3_option to incrementally extend this list of options.

TANGLEFLAGS
The flags passed to notangle (default -L'<*LINE %L %F *>%N').

WEAVEFLAGS
The flags passed to noweave (default -n).
