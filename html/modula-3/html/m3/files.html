Files

The m3 command recognizes files with these extensions.

Arguments that name directories are searched recursively. Any files with recognized suffixes (.i3, .m3, .ic, .mc, .c, .is, .ms, .s, .io, .mo, .o, .a) found in those directories are effectively added to the command line. Any directories containing .i3 files are prepended to the -D search path.
