The architecture column gives the names used throughout the SRC Modula-3 system and documentation that specify platform-specific features. If the architecture name is a link, the link points to pre-compiled binaries of SRC Modula-3 for the named platform.

You should check the page describing the known bugs and patches for the current release. There are also special installation notes for the NT386 platform that describe what tools and libraries you will need on your PC to use this release.

The status column gives the latest known status: 

    OK  - the system builds and runs out of the box
    UP  - the system is useable once patches are installed
    xx  - the system is known to have problems
    ?   - as far as we know, nobody has tried the current version
Modula-3 implementations also exist for other PC platforms, such as PC's running OS/2 or MSDOS. See the Modula-3 home page for the complete list of Modula-3 implementations.
