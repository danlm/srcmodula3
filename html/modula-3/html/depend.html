Installation dependencies

Most of the source packages included in the distribution depend on others. To successfully build one, you may need to build and install several others. Here's a list of the known dependencies: 

   archive               dependencies
   --------------        ---------------------------
   boot-XYZ              -none-
   html                  -none-

   m3utils               boot-XYZ
   m3cc                  boot-XYZ
   m3gdb                 boot-XYZ
   libm3                 boot-XYZ

   compiler              libm3
   tools                 libm3
   misc                  libm3
   m3tk                  libm3
   trestle               libm3

   netobj                m3tk

   games                 trestle
   vbtkit                trestle, tools (m3bundle)

   formsvbt              vbtkit

   vbtapps               formsvbt
   zeus                  formsvbt
   postcard              formsvbt

   mentor                zeus
   obliq                 zeus, netobj (stubgen)

   visualobliq           obliq

   pkgtools              netobj
