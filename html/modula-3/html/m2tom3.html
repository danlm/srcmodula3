m2tom3: A Modula-2 to Modula-3 translator

m2tom3 translates Modula-2 programs to Modula-3. It is available via anonymous ftp from ftp-i3.informatik.rwth-aachen.de in pub/Modula-3-Contrib/m2tom3/m2tom3-2.03.tar.gz.

Here is the latest announcement: 

From: Peter Klein <pk@i3.informatik.rwth-aachen.de>

Date: Mon, 4 Sep 1995 13:33:56 +0200 (MET DST)

----------------------------------------------------------------------

               m2tom3 Version 2.03 Release Announcement

----------------------------------------------------------------------

After some bugfixes and minor extensions, we are now able to release
version 2.03 of our Modula-2 to Modula-3 translator.

From the README:

m2tom3 is a system to port Modula-2 programs to Modula-3. It consists
of two parts:

 - A conversion program which tries to create a Modula-3 source text
   with the same semantics as the Modula-2 input while retaining the
   original look and feel as good as possible.

 - A base library which tries to emulate the Modula-2 standard library
   using the Modula-3 standard library.
