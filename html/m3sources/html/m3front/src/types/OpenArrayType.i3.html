INTERFACE OpenArrayType;

IMPORT Type, CG;

PROCEDURE New (element: Type.T): Type.T;

PROCEDURE Is (t: Type.T): BOOLEAN;
Return TRUE iff 't' is an open array type 

PROCEDURE Split (t: Type.T;  VAR element: Type.T): BOOLEAN;
If 't' is an open array type, returns TRUE and sets 'element' to the appropriate type. Otherwise, returns FALSE 

PROCEDURE EltPack (array: Type.T): INTEGER;
If 'array' is an open array type, returns the packed size in bits of the elements. Otherwise, returns 0. 

PROCEDURE EltAlign (array: Type.T): INTEGER;
If 'array' is an open array type, returns the bit alignment of the elements. Otherwise, returns Target.Byte. 

PROCEDURE OpenDepth (t: Type.T): INTEGER;
If 't' is an n-dimensional open array, returns n else returns 0 

PROCEDURE OpenType (t: Type.T): Type.T;
If 't' is an n-dimensional open array, returns the type of the base elements; otherwise, returns t. That is, strip all the ARRAY OF in front of t 

PROCEDURE DeclareTemp (t: Type.T): CG.Var;
If 't' is an open array, declare and return a temporary to hold its dope vector, otherwise abort. 

END OpenArrayType.
