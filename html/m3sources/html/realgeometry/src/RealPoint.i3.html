INTERFACE RealPoint;

 Real pairs.


   This interface implements the 2-dimensional vector space R^2. 
   It defines the basic linear operations on real pairs (arrays 
   with 2 REAL components).  See R2x2.i3 and R2IO.i3 for 
   additional operations.
