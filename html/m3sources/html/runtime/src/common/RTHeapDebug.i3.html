INTERFACE RTHeapDebug;

PROCEDURE Free(r: REFANY);
Asserts that r is not reachable. 

PROCEDURE CheckHeap();
Examines the heap, searching for objects that have been asserted free, but are reachable from a global variable, printing (to standared error) a path from that global to the object for each such putatively free object. 

CONST MaxFree = 25;
The maximum number of free objects that will be remembered. If the runtime parameter @M3heapDebugMaxFree is set to a positive integer value, that value will be used instead. 

END RTHeapDebug.
