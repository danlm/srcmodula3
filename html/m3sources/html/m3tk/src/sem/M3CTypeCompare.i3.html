IMPORT M3AST_AS, M3AST_SM;

PROCEDURE Identical(t1, t2: M3AST_SM.TYPE_SPEC_UNSET): BOOLEAN RAISES {};
are the given types identical? Also succeeds if either type is unset 

PROCEDURE Similar(t1, t2: M3AST_AS.TYPE_SPEC): BOOLEAN RAISES {};
are the given types similar? Two types are similar if they have the same form (e.g. both records with same number of fields, with the same names and defaults) and their component types have the same type code. 'Similar' is used by the type equivalence/minimisation algorithm 

END M3CTypeCompare.
