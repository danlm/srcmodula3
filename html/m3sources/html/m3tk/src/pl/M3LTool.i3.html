IMPORT M3Context, M3Args;

PROCEDURE Initialise(): M3Args.T RAISES {};
Requires explicit initialisation because it calls M3LBackEnd.RegisterArgs, which is a procedure variable initialised by a module unknown to this one. Returns the tool handle. 

PROCEDURE Run(c: M3Context.T; do_depends := TRUE): INTEGER RAISES {};
Run the pre-linker tool on 'c'. Any error returns < 0. If do_depends = FALSE, the pl_depends_on data is assumed to have been computred already. 

END M3LTool.
