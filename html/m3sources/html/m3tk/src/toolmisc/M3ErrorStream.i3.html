INTERFACE M3ErrorStream;

IMPORT Wr;

PROCEDURE Set(s: Wr.T): Wr.T;

 For an implementation of M3Error which displays errors on a stream,
this procedure redirects future output to 's', and returns the previous
stream. 

END M3ErrorStream.
