<* PRAGMA SUBTYPE *>
A ZTilps.T multi-split is like a ZSplit, except that its children are stored from bottom to top. For example, MultiSplit.Nth(v,0) returns the background child of the ZTilps. 

INTERFACE ZTilps;

IMPORT ZSplit;

TYPE
  <* SUBTYPE T <: MultiSplit.T *>
  T <: Public;
  Public = ZSplit.T OBJECT
           METHODS
             <* LL <= VBT.mu *>
             init (saveBits := FALSE; parlim := -1): T
           END;
The call v.init(...) initializes v as a ZTilps and returns v. See the ZSplit interface for a description of saveBits and parlim. 

END ZTilps.
