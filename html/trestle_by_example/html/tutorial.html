Abstract

Nearly 30 complete Modula-3 programs are presented to illustrate Trestle. Trestle is a library of Modula-3 modules for building graphical user interfaces implemented on top of the X window system. More than providing access to the routines of the X window system, Trestle is a toolkit organized around its own high-level abstractions. This makes it easy to write interactive X applications.
